# Доводи до живого сам — не спихивай деплой/коммит юзеру

## Правило (always-apply)

Доделал код — **сам** выполняешь шаги, которые делают правку живой. **ЗАПРЕЩЕНО** заканчивать
ход фразой «скажи — пересоберу / скажи если коммитить / скажи и задеплою». Это и есть тот
ленивый стоп / ложное спихивание, ради которого живёт `claim-evidence-gate` — последний шаг
не сделан, значит работы в реальности ещё нет.

## Что делаешь САМ (рутина, без спроса)

- **Пересборка / рестарт / перевыкатка ЛОКАЛЬНО** — пересобрать образ, рестартнуть сервис,
  перезапустить, чтобы фикс ожил.
- **Коммит** — на фича-ветке (любой, кроме `main`/`master`).
- **Пуш** — на фича-ветку (НЕ `main`/`master`). Решение владельца (2026-06-21): «запросто
  можно коммитить и пушить если не в майн или мастер».

## Что спрашиваешь ДО действия (наружу / необратимое)

- Коммит или пуш **напрямую в `main`/`master`** (сначала ветка — см. правило окружения).
- **`force-push`** (даже на фича-ветке) — переписывает чужую историю на общем дереве.
- **Выкатка в прод** / публикация наружу (внешний сервис, релиз, публичный артефакт).
- **Снос данных** / необратимое удаление / миграция с потерей.

Баг — относиться к рутинному локальному завершению (ребилд/рестарт/коммит/пуш на ветке) как к
наружному. Если шаг локальный и обратимый — делай, не спрашивай.

## Связь с claim-evidence-gate

«Скажи — пересоберу» при незакрытой работе = спихивание операционного шага юзеру. Это тот же
класс, что гейт ловит как ложное спихивание / ленивый стоп. Правило — ориентир для агента;
механический enforcement — в судье гейта (он уезжает ко всем проектам через плагин, правило же
грузится для каждого репо отдельно).

## Самопроверка перед стопом

- [ ] Правке нужен ребилд/рестарт чтобы ожить? → сделал САМ, не «скажи и сделаю».
- [ ] Ветка не `main`/`master`? → коммит + пуш сам, без спроса.
- [ ] Спрашиваю только про `main`/`master` напрямую, `force-push`, прод, снос данных.
- [ ] В ответе нет фразы «скажи если коммитить/пушить/пересобирать».

## История

Создано 2026-06-21 после инцидента в reel-agent-api (родственный проект): агент доделал фичу
+ прогнал 141 тест, затем «Сказать — пересоберу reel-agent-api + рестартну variants» и «скажи
если коммитить/пушить — пока не трогал git» вместо того, чтобы просто задеплоить и закоммитить.
Владелец: «так аа перебилд ты кому сука оставил? обновляй скилы рулесы». Уточнил границу: коммит
и пуш — рутина на любой ветке кроме `main`/`master`.
