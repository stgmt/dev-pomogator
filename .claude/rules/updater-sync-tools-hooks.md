# Апдейтер должен синхронизировать tools и hooks

Если апдейтер обновляет только commands, функционал расширений может остаться старым (tools/hooks не обновлены).

## Правильно

- Обновлять commands по манифесту
- Обновлять toolFiles по манифесту
- Переустанавливать hooks для projectPaths

## Неправильно

- Обновлять только `.cursor/commands`
- Игнорировать `.dev-pomogator/tools/` и `hooks` при апдейте

## Чеклист

- [ ] Есть `toolFiles` в манифесте расширения
- [ ] Апдейтер пишет tools в `projectPath`
- [ ] Апдейтер ставит/обновляет hooks после апдейта
