# Апдейтер синхронизирует ВСЕ артефакты плагинов

Апдейтер обновляет ВСЕ установленные плагины dev-pomogator целиком: commands, rules, tools, skills, hooks. Если обновить только часть — функционал расширений рассинхронизируется.

## Правильно

- Обновлять commands по манифесту
- Обновлять rules по манифесту
- Обновлять toolFiles по манифесту
- Обновлять skillFiles по манифесту
- Переустанавливать hooks для projectPaths

## Неправильно

- Обновлять только commands без tools/hooks
- Игнорировать `.dev-pomogator/tools/` при апдейте
- Игнорировать `.claude/skills/` при апдейте
- Оставлять старые hooks после обновления extension

## Чеклист

- [ ] Есть `toolFiles` в манифесте расширения
- [ ] Есть `skillFiles` в манифесте расширения (если skills определены)
- [ ] Апдейтер пишет tools в `projectPath`
- [ ] Апдейтер пишет skills в `projectPath`
- [ ] Апдейтер ставит/обновляет hooks после апдейта
