Команда `/answer-simple` была вызвана без аргументов — нечего обрабатывать.

Использование:

```
/answer-simple <черновик ответа>
```

Что делает: аудит черновика ответа на жаргон, внутренние коды, multi-select с >3 опциями и отсутствие микроистории. Возвращает блоки:

- **Переформулировано:** — отредактированная версия
- **Найдено проблем:** — список выявленных issues

Пример:

```
/answer-simple Привет! По вашему тикету INC-12345 эскалирую L2, SLA P1 истекает через 4h. Выберите: 1) hotfix, 2) rollback, 3) workaround, 4) defer.
```

Передайте черновик аргументом и повторите вызов.
