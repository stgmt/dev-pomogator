# User Stories

- Как пользователь dev-pomogator, я хочу чтобы claude-mem устанавливался автоматически и полностью (worker + chroma + MCP + health hooks), чтобы persistent memory работала сразу после установки без ручных шагов.
- Как пользователь dev-pomogator, я хочу чтобы при сбое установки claude-mem я видел конкретную причину (какой шаг, что пошло не так, как починить), чтобы не копаться в логах вручную.
- Как пользователь dev-pomogator, я хочу чтобы после установки автоматически проверялось что все сервисы живы (worker health, chroma heartbeat, MCP доступен), чтобы не узнавать о проблемах при первом использовании.
- Как пользователь dev-pomogator, я хочу чтобы claude-mem-health хуки ставились автоматически при установке suggest-rules, чтобы chroma авто-рестартился при каждом SessionStart.
