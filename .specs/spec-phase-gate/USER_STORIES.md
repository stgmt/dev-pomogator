# User Stories

- Как пользователь dev-pomogator, я хочу чтобы Claude физически не мог записывать файлы будущих фаз спеки до подтверждения текущей СТОП-точки, чтобы я мог проверить промежуточные результаты перед продолжением. @feature1
- Как пользователь dev-pomogator, я хочу видеть текущую фазу и список разрешённых/запрещённых файлов в начале каждого промпта, чтобы понимать состояние workflow без запуска скриптов. @feature2
- Как пользователь dev-pomogator, я хочу чтобы audit-spec.ps1 обнаруживал partial implementation (task [x] при "НЕ РЕАЛИЗОВАНО" в FR), чтобы предотвратить маскировку незавершённой работы. @feature3
- Как пользователь dev-pomogator, я хочу чтобы правила specs-management.md требовали decomposition FR с несколькими вариантами реализации, чтобы каждый вариант имел свой AC, BDD сценарий и задачу. @feature4
