# User Stories

## US-1: Подсказка следующего шага @feature1

Как разработчик, я хочу получать подсказку о следующем шаге после завершения сессии Claude Code, чтобы быстро продолжить работу не теряя контекст.

## US-2: Визуальная заметность подсказки @feature9

Как разработчик, я хочу видеть подсказку с 💡 emoji сразу после остановки агента, чтобы быстро оценить и решить — нажимать `+` или нет.

## US-3: Активация подсказки через "+" @feature2

Как разработчик, я хочу активировать подсказку вводом `+`, чтобы Claude выполнил предложенное действие без необходимости набирать полную команду.
