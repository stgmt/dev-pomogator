# User Stories

- Как разработчик плагина, я хочу пометить расширение как beta, чтобы пользователи знали что оно экспериментальное. @feature1
- Как пользователь, я хочу видеть список плагинов с их статусом (stable/beta) при установке, чтобы принимать осознанное решение. @feature2
- Как пользователь, я хочу чтобы `--all` не устанавливал beta плагины по умолчанию, чтобы не получить нестабильный функционал без явного выбора. @feature3
- Как пользователь, я хочу явно установить beta плагин (`--include-beta` или выбором в checkbox), чтобы протестировать новый функционал. @feature4
