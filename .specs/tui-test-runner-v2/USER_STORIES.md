# User Stories

- Как разработчик, я хочу видеть AI-анализ провалившихся тестов с подсказками по исправлению, чтобы быстрее находить корневую причину ошибок. @feature1
- Как разработчик, я хочу кликать по путям к файлам в логах тестов, чтобы мгновенно переходить к нужному месту в коде. @feature2
- Как разработчик, я хочу видеть список всех тестов до запуска и выбирать какие запустить, чтобы экономить время на отладке конкретных сценариев. @feature3
- Как разработчик, я хочу чтобы TUI запоминал мой последний таб, фильтры и развёрнутые узлы между сессиями, чтобы не настраивать рабочее пространство каждый раз. @feature4
- Как разработчик, я хочу настраивать паттерны ошибок через YAML-файл, чтобы расширять автоматическую категоризацию под свой проект. @feature5
- Как разработчик, я хочу запускать TUI по комбинации клавиш в Claude Code с автозапуском тестов, чтобы одним нажатием видеть прогресс. @feature6
- Как разработчик, я хочу экспортировать скриншот TUI в SVG для шаринга и отчётов. @feature7
