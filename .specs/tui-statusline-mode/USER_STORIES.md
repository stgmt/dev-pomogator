# User Stories

- Как разработчик, я хочу видеть прогресс тестов в компактном виде (3-5 строк) в нижнем пейне Windows Terminal, чтобы не переключаться на полный TUI во время работы в Claude Code. @feature1
- Как разработчик, я хочу переключаться между compact и full режимами TUI по кнопке/клавише, чтобы быстро получить детали при ошибке и вернуться к компактному виду. @feature2
- Как разработчик, я хочу останавливать тесты кнопкой/клавишей прямо из TUI, чтобы не искать PID и не переключаться в другой терминал. @feature3
- Как разработчик, я хочу чтобы TUI автоматически переключался в compact mode при маленьком размере пейна, чтобы не крэшился при ресайзе. @feature4
- Как разработчик, я хочу чтобы Claude Code statusline слот освободился от test-statusline рендера, чтобы использовать его для своих нужд. @feature5
