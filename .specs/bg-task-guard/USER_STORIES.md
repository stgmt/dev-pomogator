# User Stories

- Как разработчик, я хочу чтобы Claude Code автоматически продолжал работу после запуска фоновых задач (Docker тесты, долгие билды), чтобы не терять 7-12 минут простоя на каждый прогон. @feature1
- Как разработчик, я хочу чтобы маркер фоновой задачи автоматически истекал через 15 минут, чтобы зависший marker не блокировал Claude навечно. @feature2
